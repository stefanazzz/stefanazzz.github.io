There are 2 options:

I. link to pdf file:

  1. copy the pdf into the subfolder "files" of the tiddy wiki folder (~/knowledgewiki/files)
  2. in the tiddler where you wish to show the pdf, make this link:

If there are puny white spaces in the file name, replace them with %20

II. Direct "binary" content inside the tiddle:

  1. Drag-drop pdf into tiddly workspace (not in a tiddly).
  2. A tiddle dialog dialog opens: click on "Import".
  3. The tiddle dialogue now shows a link to the file name: click on the filename.
  4. Another tiddle opens, previewing the pdf:
    1. click edit
    2. modify the title of this new tiddle to what you want
    3. save (confirm changes).
  5. You can now delete all previous tiddles, and just keep the one with the edited title.
